数据收集

AI Pen 扩展不收集任何个人身份信息、浏览历史或使用数据。我们重视您的隐私。

API 使用

您选中的文本会发送到 DeepSeek API 进行AI处理。API Key 存储在您本地的 Chrome Storage 中，仅用于调用 DeepSeek 接口。

文本内容经 DeepSeek API 处理后返回结果即丢弃。我们不保存、不记录、不上传任何文本内容。

存储

仅存储以下数据在您本地浏览器：API Key、每日使用计数、最近的改写历史记录。

权限

  • activeTab：读取当前页面选中的文字
  • storage：保存设置和使用记录
  • host_permissions：调用 DeepSeek API
